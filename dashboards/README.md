# Dashboards

The Eidam dashboard system uses a dark theme with Mushroom Cards as the primary framework.

## Required HACS Components

| Component | Required | Purpose |
|-----------|----------|---------|
| [Mushroom Cards](https://github.com/piitaya/lovelace-mushroom) | Yes | Primary card framework |
| [card-mod](https://github.com/thomasloven/lovelace-card-mod) | Yes | Custom CSS styling |
| [mini-graph-card](https://github.com/kalkih/mini-graph-card) | Optional | Temperature/energy graphs |
| [navbar-card](https://github.com/joseluis9595/lovelace-navbar-card) | Yes | Bottom navigation bar |
| [bubble-card](https://github.com/Clooos/Bubble-Card) | Optional | Popup panels |
| [stack-in-card](https://github.com/custom-cards/stack-in-card) | Optional | Card grouping |

## Views

| View | Description |
|------|-------------|
| Home | Overview with weather, room status, quick actions |
| Security | Ring cameras, alarm panel, door sensors |
| Climate | Tado zones, temperature graphs |
| Energy | Per-room power monitoring, daily costs |
| Lighting | All lights with room grouping |
| Lounge | Room-specific controls |
| Bedroom | Room-specific controls |
| Lucas Room | Baby monitor, routines, environment |
| Kitchen & Office | Combined view |
| Bathrooms | Sensor-triggered controls |
| Appliances | Miele, Bosch, coffee machine status |
| Media | Sonos, TVs, streaming |
| Settings | Helper toggles, system status |
